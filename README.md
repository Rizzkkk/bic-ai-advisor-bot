# BIC AI Advisor Bot

## Overview
The BIC AI Advisor Bot is an intelligent chat interface that connects users with Bibhrajit, a seasoned founder with extensive experience in AI, robotics, and autonomy. The platform provides strategic business advice and mentorship through an interactive chat interface.

## Features
- Real-time chat with AI-powered responses
- Investment guidance and business advisory services
- Secure and user-friendly interface

## Getting Started
1. Clone the repository.
2. Install dependencies with `npm install`.
3. Set up your environment variables.
4. Run the application with `npm start`.
